# HR-Analytics_Dashboard
## Project Overview

This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. The dashboard helps analyze employee attrition by different factors such as age, education, salary, job role, gender, and years at the company.

The objective of this project is to identify patterns in employee attrition and provide insights that can help HR teams make better workforce decisions.

## Tools & Technologies Used

- Microsoft Power BI
- Microsoft Excel / CSV Dataset
- Power Query
- DAX
- Data Visualization
- 
## Dataset
The dashboard is created using the HR Analytics Dataset containing employee-related information such as:

- Employee ID
- Age
- Gender
- Department
- Education
- Job Role
- Salary Slab
- Years at Company
- Attrition Status
- Monthly Income
- Job Satisfaction

- Total Employees : **1.47K**

- Average Salary : **6.50K**


## Dashboard Visualizations

The dashboard includes:

- KPI Cards
- Attrition by Education (Donut Chart)
- Attrition by Age (Column Chart)
- Attrition by Gender (Treemap)
- Attrition by Salary Slab (Bar Chart)
- Attrition by Years at Company (Area Chart)
- Attrition by Job Role (Bar Chart)
- Job Role Summary Table
- Department Slicer


##  Key Insights

- Employees aged **26–35 years** have the highest attrition.
- **Laboratory Technicians** experience the highest attrition among job roles.
- Employees earning **up to 5K salary** show the highest attrition.
- Employees from the **Life Sciences** education field contribute the largest share of attrition.
- Attrition decreases as employee tenure increases.
- Department slicer allows interactive filtering across the dashboard.
